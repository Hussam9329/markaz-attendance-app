# Markaz HR Pro — برنامج حضور ورواتب الموظفين

نظام Next.js احترافي لإدارة ملفات الموظفين، تسجيل الحضور عبر QR، حساب التأخير والغياب، احتساب الرواتب الشهرية، وتصدير كشف الرواتب CSV.

---

## الميزات الرئيسية

- 📱 واجهة تابلت/موبايل لمسح QR وتسجيل الحضور فورياً.
- 🔐 لوحة إدارة منفصلة بكلمة مرور.
- 👥 ملفات موظفين كاملة: كود موظف، الاسم، القسم، الوظيفة، الهاتف، تاريخ المباشرة، ملاحظة الدفع، الراتب الأساسي، المخصصات، وحالة التفعيل.
- 🧾 توليد QR تلقائي لكل موظف مع إمكانية إعادة التوليد.
- 📋 سجل حضور يومي يوضح الحاضرين، المتأخرين، والغائبين.
- ⏰ احتساب التأخير حسب وقت بداية التأخير ومبلغ الخصم لكل دقيقة.
- 🚫 احتساب الغياب من أيام العمل الشهرية المحددة في الإعدادات.
- 💰 كشف رواتب شهري: الراتب الأساسي، المخصصات، إجمالي المستحق، خصم التأخير، خصم الغياب، إجمالي الخصومات، وصافي الراتب.
- 📊 مركز تقارير إداري ومالي مع ملخصات للحضور والرواتب.
- 📥 تصدير كشف الرواتب بصيغة CSV جاهز للمحاسبة والأرشفة.
- 🗄️ قاعدة بيانات Neon PostgreSQL.

---

## التشغيل المحلي

```bash
cp .env.example .env.local
npm install
npm run dev
```

ثم افتح:

- واجهة المسح: `http://localhost:3000`
- لوحة الإدارة: `http://localhost:3000/admin/login`

---

## إعداد قاعدة البيانات

1. أنشئ قاعدة بيانات Neon من neon.tech.
2. افتح Neon SQL Editor.
3. شغّل محتوى الملف:

```bash
db/schema.sql
```

> مهم: إذا كان لديك إصدار قديم من قاعدة البيانات، شغّل نفس ملف `schema.sql` أيضاً لأنه يحتوي أوامر `ALTER TABLE ... ADD COLUMN IF NOT EXISTS` لإضافة حقول الموظفين الجديدة مثل القسم والهاتف والمخصصات.

## متغيرات البيئة

انسخ `.env.example` إلى `.env.local` وضع القيم التالية:

| المتغير | الوصف |
|---------|-------|
| `DATABASE_URL` | رابط اتصال Neon PostgreSQL |
| `ADMIN_PASSWORD` | كلمة مرور لوحة الإدارة |
| `AUTH_SECRET` | مفتاح تشفير الجلسة |
| `NEXT_PUBLIC_APP_URL` | رابط التطبيق بعد النشر |

## قواعد احتساب الراتب

من صفحة **الإعدادات** يمكنك تحديد:

- اسم المركز.
- المنطقة الزمنية، والافتراضي `Asia/Baghdad`.
- العملة، مثل `IQD`.
- وقت بداية التأخير.
- مبلغ الخصم لكل دقيقة تأخير.
- أيام العمل الشهرية.

كشف الرواتب يحسب:

```text
إجمالي المستحق = الراتب الأساسي + المخصصات
خصم الغياب = الراتب الأساسي ÷ أيام العمل الشهرية × أيام الغياب
صافي الراتب = إجمالي المستحق - خصم التأخير - خصم الغياب
```

في الشهر الحالي، يحسب النظام أيام العمل المتوقعة حتى اليوم الحالي فقط، حتى لا يخصم بقية الشهر قبل أوانه.
